# examyuan 考试猿

---
&nbsp;&nbsp;One exam project named examyuan!  
&nbsp;&nbsp;这是一个叫做"考试猿"的在线考试系统,本系统旨在帮助中小学校、机构、团队解决在线考试的问题。  

---
### 预计功能列表：
* 用户功能
    * 用户登录
    * 个人页面
    * 答题
    * 答题记录
* 后台管理功能
    * 登录
    * 管理老师账号功能
    * 导入学生信息
    * 导入分类功能
    * 导入题目及答案功能
    * 导入知识点功能
    * 导出学生做题信息
    * 导出错题相关信息
